# cjpp
JavaScript Compiler

# Requirements
Flex, Bison, and a C++ compiler are required

# Running

Use the Make file to compile the source code.

Using terminal, enter:

    cjpp < 'file_to_compile'
    
to compile your JavaScript. It will be output as a C file named 'output.c'
